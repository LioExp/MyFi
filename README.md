# MyFi
MyFi é um monitor prático de redes para MiFi desenvolvido em Python. Descobre dispositivos conectados (IP, MAC), regista logs, controla banda e oferece automação modular (chunks). Inclui soft delete, deteção de duplicatas por hash e planeia integração com IA para a deteção de anomalias. Projeto open‑source para gestão e segurança de redes locais.
